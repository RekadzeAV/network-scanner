# Release Acceptance Checklist

Финальная проверка перед выпуском версии.

## 1) Репозиторий и документация

- [ ] `CHANGELOG.md` обновлен (раздел `Unreleased` или целевая версия).
- [ ] `README.md` отражает актуальные возможности и ссылки на новые документы.
- [ ] Обновлены пользовательские документы:
  - [ ] `docs/USER_GUIDE.md`
  - [ ] `docs/GUI_SMOKE_CHECKLIST.md`
  - [ ] `docs/RELEASE_SUMMARY_UI_RESULTS.md`
  - [ ] `docs/PR_DESCRIPTION_UI_RESULTS.md`

## 2) Автотесты и качество

- [ ] Выполнен `go test ./...` без ошибок.
- [ ] Нет новых проблем линтера в измененных файлах.
- [ ] Проверены ключевые unit-тесты модели результатов:
  - сортировка;
  - фильтрация;
  - нормализация типов;
  - лимит/формат чипов портов.

## 3) Сборка артефактов

- [ ] CLI собирается: `go build -o network-scanner ./cmd/network-scanner`
- [ ] GUI собирается: `go build -o network-scanner-gui ./cmd/gui`
- [ ] При необходимости выполнены smoke-скрипты CLI:
  - `scripts/smoke-cli-no-topology.sh|ps1`
  - `scripts/smoke-cli-topology.sh|ps1`

## 4) GUI приёмка (ручная)

- [ ] Пройден `docs/GUI_SMOKE_CHECKLIST.md`.
- [ ] Проверено сохранение/восстановление:
  - режима (`Таблица`/`Карточки`);
  - сортировки;
  - фильтров;
  - лимита чипов.
- [ ] Проверено, что `Сохранить результаты` экспортирует текущее представление (фильтры + сортировка).

## 5) Регрессии топологии

- [ ] Вкладка `Топология` открывается корректно.
- [ ] Построение топологии отрабатывает без ошибок.
- [ ] Превью PNG/масштабирование/сохранение работают.

## 6) Релизные артефакты

- [ ] Подготовлен текст релиз-нотов (можно использовать `docs/RELEASE_SUMMARY_UI_RESULTS.md`).
- [ ] Подготовлено описание PR (можно использовать `docs/PR_DESCRIPTION_UI_RESULTS.md`).
- [ ] Проверена версия/дата в релевантных документах.
