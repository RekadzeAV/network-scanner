## 🎯 Основные изменения

### ✅ Исправления в скриптах сборки

#### Критические исправления:
1. **Добавлен `CGO_ENABLED=1` для всех GUI сборок**
   - Исправлен `scripts/build.sh`: добавлен CGO для Linux и macOS GUI сборок
   - Исправлен `scripts/build-macos.sh`: добавлен CGO для всех GUI сборок
   - Fyne framework требует CGO для работы, без него GUI версии не собирались корректно

2. **Улучшен скрипт `build-macos.sh`:**
   - Добавлено создание universal binary для GUI версии (в дополнение к CLI)
   - Улучшена обработка ошибок - скрипт продолжает работу при ошибках отдельных версий
   - Добавлены проверки существования файлов перед копированием
   - Улучшен вывод результатов сборки

3. **Исправлен `go.mod`:**
   - `github.com/jung-kurt/gofpdf/v2` теперь прямая зависимость (используется в `cmd/md2pdf`)
   - Устранено предупреждение линтера

## 📦 Сборка релиза

Для создания релиза используйте соответствующие скрипты:

### macOS:
```bash
./scripts/build-macos.sh
```

### Linux/Unix (все платформы):
```bash
./scripts/build.sh
```

### Windows:
```powershell
.\scripts\build.bat
```

Релизные скрипты складывают артефакты в **`build/release/<YYYY-MM-DD-N>/`** (на Windows — в подкаталоге **`windows\`**); см. `docs/BUILD_STRUCTURE.md`.

## ⚠️ Важные замечания

1. **CGO обязателен для GUI версий:**
   - Все GUI сборки теперь явно используют `CGO_ENABLED=1`
   - Убедитесь, что у вас установлен C компилятор (GCC, clang и т.д.)

2. **macOS сборка:**
   - Скрипт `build-macos.sh` теперь создает universal binary для GUI версии
   - Требуется Xcode Command Line Tools для создания universal binary (lipo)

3. **Зависимости:**
   - Все зависимости обновлены и проверены
   - `go mod tidy` выполнен для очистки зависимостей
