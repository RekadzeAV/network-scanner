# Legacy Directory

Этот каталог содержит устаревшие файлы, которые были архивированы в процессе рефакторинга.

## Содержимое

- `GUI_STRUCTURE.txt` - Устаревшая ASCII-диаграмма UI (заменена на актуальную документацию в `docs/GUI.md`)
- `launch-gui.sh` - Скрипт запуска GUI на macOS (дублирует `scripts/build-gui-release.sh`)
- `build-release-windows-only.ps1` - Скрипт сборки Windows (дублирует `scripts/build-windows.sh`)

## Почему они здесь?

Эти файлы были удалены из корня проекта, так как:
1. Они дублируют функциональность существующих скриптов в `scripts/`
2. Ссылки на них обновлены в документации
3. Они сохранены для исторической справки

## Как использовать

Если вам нужен один из этих файлов для особых целей, скопируйте его из этого каталога.
В новых сборках используйте актуальные скрипты из `scripts/`.
